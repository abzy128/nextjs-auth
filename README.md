# Auth demo

## Installation

1. Clone the repo

```bash
git clone https://github.com/abzy128/nextjs-auth.git
```

2. Install dependencies:

```bash
yarn install
```

3. Add your MongoDB URI to `.env.local` (you'll need to create the file)

```text
MONGO_URI=YOUR_MONGO_URI
```

4. Run the server

```bash
yarn build && yarn start
```
